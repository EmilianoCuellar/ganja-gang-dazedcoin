# Ganja Gang — Investor Landing (Next.js 14 + Tailwind)

A polished investor- and pilot-focused landing page highlighting PoP + zk‑ID, KPIs, and the revenue model.

## Quickstart

```bash
pnpm i   # or: npm i / yarn
pnpm dev # or: npm run dev / yarn dev
```

Then open http://localhost:3000

## Tech
- Next.js 14 (App Router)
- TailwindCSS
- Framer Motion
- Lucide Icons

## Customize
- Replace `/public/deck.pdf` with your real deck.
- Swap hero visuals with screenshots in the Hero card.
- Update copy in `app/page.tsx` (Economics, KPIs, Roadmap, Team).
- Add pages for `/privacy`, `/terms`, `/docs` if needed.

## Deploy
- Vercel (zero config) or Netlify.
